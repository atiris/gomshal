Gomshal library demo

  1. Initialize gomshal: Gomshal instance
  2. Set custom configuration gConfiguration: GomshalConfiguration = JSON.parse(...) and call initialize.



    If response state is not GomshalState.Ok you can change gInputs and call initialize again.
    If 2FA is required, approve login on phone and then call sharedLocations with empty gInputs.
  3. Get shared locations
  4. Close page and browser gomshal.close() and set gomshal = undefined;
