Gomshal library demo

  1. Initialize gomshal: Gomshal instance
  2. Set custom configuration gConfiguration: GConfiguration = JSON.parse(...) and call initialize.
    If response state is not GStep.LocationData you can check gomshal.error change gConfiguration and call initialize again.
    If 2FA is required, approve login on phone and then call initialize with empty gConfiguration input attribute.
  3. Get last detected shared locations, set or remove callback when new shared locations data detected.
  4. Close GoogleMaps page and browser gomshal.close() and set gomshal = undefined;
