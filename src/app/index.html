Gomshal library demo

  1. Initialize gomshal: Gomshal instance
  2. Set custom configuration gConfiguration: GConfiguration = JSON.parse(...) and call initialize. Login and password are cached.
    After logging in, you do not have to enter login and password here in the configuration the next time you call initialize.
    If response state is not GState.LocationData you can check gomshal.error change gConfiguration and call initialize again.
    If 2FA is required, approve login on phone and then call initialize with empty gConfiguration input attribute.
  3. Get last shared locations, and set or remove short callback when new locations detected.
  4. Close GoogleMaps page and browser gomshal.close() and set gomshal = undefined;
