Gomshal library demo

  1. Create gSettings: GomshalSettings object from JSON

  2. Initialize gomshal: Gomshal instance with gSettings settings
  3. Create (or replace last) gInputs: GomshalInputs object from JSON

  4. Call location with actual inputs gomshal.getLocations(gInputs)

    If response is not success you can change i in previous step and call getSharedLocations again.
    If 2FA is required, allow login on phone and then call getSharedLocations with empty gInputs.
  5. Close browser, clean everything with gomshal.close()
  • Only open browser gomshal.startBrowser() (please initialize gomshal: Gomshal before call this)
